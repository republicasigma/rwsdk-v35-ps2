# RenderWare "Graphics" 3 (3.5), PlayStation 2 — Evaluation Edition
Uma distro, edição de avaliação, da versão 3.5 para [PlayStation 2](https://sigmaco.org/t/playstation-2) do SDK de [RenderWare](https://sigmaco.org/t/renderware) Graphics, um [middleware](https://sigmaco.org/t/middleware) adjunto de plugins.
Atualmente, este repositório compreende uma conjuntura de código pré-compilado em [C](https://sigmaco.org/t/c), devido a ser uma edição de avaliação.

Para quaisquer dúvidas ou dificuldades, use nosso canal de suporte no [Discord](https://sigmaco.org/t/discord):
